# Weekly Growth Intelligence Agent

Automated AI agent that monitors GA4 channel performance weekly,
detects anomalies, and delivers a plain-English brief + live dashboard.

## Architecture
Schedule → BigQuery → Code (anomaly detection) → Groq LLM → Gmail
                          ↓
                    Google Sheets → Looker Studio Dashboard

## Stack
- BigQuery — unsampled GA4 raw data
- n8n (self-hosted Docker) — workflow automation
- Groq API (Llama 3.3 70B) — AI brief generation
- Google Sheets — structured data storage
- Looker Studio — live stakeholder dashboard
- Gmail SMTP — automated email delivery

## What it detects
- Week-over-week purchase changes per channel
- Anomalies above 20% threshold flagged automatically
- Root cause hypothesis generated by AI
- Recommended action included in every brief

## Key Finding (January 2021, GA4 Sample Dataset)
Organic channel dropped -25.9% in Week 4.
Direct was the only growing channel.
Paid search delivered only 37 purchases across the full month
despite being the only channel with direct cost.
